Topics

The forum program will include keynote speeches, oral and poster sessions of contributed papers. The thematic topics of 14th IFTC is Networked New Media Cooperative Innovation. Prospective authors are invited to submit manuscripts on topics including but not limited to:

3. Deep Learning for Social Media

5. Source Coding (HEVC, AVS, etc.)

9. Video Retrieval and Content Security

2. Graphics and Image Processing for Virtue Reality

4. Big Data for Next Generation TV

6. Advanced Channel Coding

8. Mobile Multimedia Communication

10. Video Analysis and Video Surveillance
